# Grafana – Copilot Context

## Purpose
Visualizes real-time and historical trading data.

## Data Sources
- InfluxDB (primary)

## Dashboards
- Live price charts
- Indicator overlays
- Buy/Sell signal markers
- Strategy performance metrics
- System latency and throughput

## Panel Types
- Time-series graphs
- Tables
- Heatmaps

## Design Guidelines
- Low refresh interval (real-time)
- Clear signal visualization
- Symbol-based filtering
- Strategy comparison support
